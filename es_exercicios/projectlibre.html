Introdução ao Project Libre

Alternativa Livre ao Microsoft Project

O que é o Project Libre?

  • Software de gerenciamento de projetos de código aberto
  • Alternativa gratuita ao Microsoft Project
  • Mais de 5 milhões de downloads worldwide
  • Compatível com formatos do Microsoft Project (MPP)
  • Disponível para Windows, macOS e Linux

Por que usar em Engenharia de Software?

  • Planejamento visual de projetos de software
  • Controle de prazos e dependências
  • Alocação de recursos da equipe
  • Acompanhamento do progresso
  • Geração de relatórios para stakeholders
Instalação do Software

Baixe e instale o Project Libre a partir do site oficial: https://www.projectlibre.com

Definindo Dependências entre Tarefas

Estabelecendo Relações e Sequências

Tipos de Dependências

  • Término-Início (FS): Mais comum - a tarefa B só inicia quando a A termina
  • Início-Início (SS): A tarefa B inicia quando a A inicia
  • Término-Término (FF): A tarefa B termina quando a A termina
  • Início-Término (SF): Menos comum - a tarefa B termina quando a A inicia

Como Estabelecer Dependências

  • Selecione duas tarefas (Ctrl + clique)
  • Clique no botão "Link Tasks" ou pressione F2
  • Arraste do centro de uma tarefa para outra no gráfico de Gantt
  • Duplo-clique na seta de dependência para alterar o tipo
Prática: Criando Dependências

Crie dependências entre as tarefas do seu projeto. Experimente diferentes tipos de dependências e observe como afetam o cronograma.

Atividade Avaliativa (1,0 Ponto)

Elaborando um Cronograma de Projeto de Software

Descrição da Atividade

Utilizando o software Project Libre, elabore o cronograma de um projeto de desenvolvimento de um sistema de e-commerce. O projeto deve conter:

  • EAP: Estrutura hierárquica com no mínimo 4 fases principais
  • Tarefas: Pelo menos 3 tarefas específicas por fase com durações estimadas
  • Recursos: No mínimo 3 recursos alocados às tarefas
  • Dependências: Relações lógicas entre as tarefas
  • Caminho Crítico: Identificação do caminho crítico do projeto

Formato de Entrega

  • Arquivo: Salve como "Cronograma_e-commerce_[SeuNome].xml"
  • Data: Conforme estipulado no Google Classroom
  • Valor: 1,0 ponto
Observações Importantes

Certifique-se de que todas as dependências estão corretamente configuradas e que o caminho crítico está visível no gráfico de Gantt.
