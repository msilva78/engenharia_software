O que são Requisitos?

Definição Formal

Requisitos são descrições dos serviços que um sistema deve fornecer e as restrições sob as quais deve operar.

Eles definem o que o sistema deve fazer.

Engenharia de Software - Aula 2

A Importância dos Requisitos

A Base de um Projeto de Sucesso

  • Base para planejamento e estimativas
  • Guia para desenvolvedores e testadores
  • Critério para validação e aceitação
  • Redução de retrabalho e custos
  • Alinhamento de expectativas

Engenharia de Software - Aula 2

A Importância em Números

Dados Estatísticos

Estudos mostram que problemas relacionados a requisitos são a causa principal de falhas em projetos de software, representando até **60%** dos problemas identificados.

Classificação de Requisitos

O que o sistema faz vs. Como ele opera

Funcionais

Descrevem **o que** o sistema deve fazer - as funcionalidades.

  • Processar pagamentos
  • Cadastrar usuários
  • Emitir relatórios

Não Funcionais

Descrevem **como** o sistema deve operar - as restrições.

  • Segurança
  • Desempenho
  • Usabilidade
